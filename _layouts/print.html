{%- comment -%} ============================================================================= print.html — a stripped-down layout for the full-yearbook print/PDF view ============================================================================= Used by print.md (/print/). This layout intentionally omits the sticky site header, the search box, and the footer so that the printed/exported PDF contains only yearbook content. A floating "Print / Save as PDF" button is shown on screen but is hidden when the browser's print dialog is open (via @media print in main.css). Book-printing specifications met by this layout + its companion CSS: • Page size : US Letter (8.5 × 11 in) • Margins : 0.75 in all sides (safe zone for most print services) • Gutter : additional 0.25 in on the binding side (inner margin = 1 in) • Color mode : RGB (sRGB) — accepted by Lulu, Blurb, and most print-on- demand services; you can convert to CMYK in Acrobat if required by your chosen service • Resolution : images are printed at whatever resolution they are stored; use 300 dpi photos for best results • Font embed : browsers embed font outlines when printing to PDF, so all text is preserved faithfully even if the reader's device does not have the web fonts installed ============================================================================= {%- endcomment -%} {%- include head.html -%} {%- comment -%} On-screen only: floating action bar with print button {%- endcomment -%}
