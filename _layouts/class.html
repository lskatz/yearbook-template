Expected front-matter fields: title — "4th Grade — Krelboynes" (required) grade — 4 (required, used for sorting on the home page) room — "Room 11" (optional) teachers — list of person IDs (optional) students — list of person IDs (optional, but c'mon) thumbnail — path to a class photo shown at the top of this page (optional) e.g. "/assets/images/classes/4th-grade-krelboynes.jpg" Falls back to `class_thumbnail_default` from _config.yml when not set, so every class page always has an image. Set `class_thumbnail_default` in _config.yml to change the fallback for every class at once. thumbnails — list of photo paths for multiple class photos (optional) When set, overrides `thumbnail` and shows all listed images in order at the top of the page. e.g.: thumbnails: - "/assets/images/classes/photo1.jpg" - "/assets/images/classes/photo2.jpg" To customize: • Section labels ("Teachers", "Students") → edit labels in _config.yml. • Add a new optional field (e.g. "motto") → add it to the Markdown file's front-matter, then reference it here as {{ page.motto }}. • Reorder sections → drag the
blocks into a new order. • Remove the teachers or students section → delete its block. ============================================================================= {%- endcomment -%}
